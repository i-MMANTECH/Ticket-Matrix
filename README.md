# Ticket Matrix — Nativetalk Ticketing Module

Full-stack ticketing module submitted as part of the Tech4mation Ltd internship evaluation. Built with **Next.js (App Router, TypeScript) + Tailwind CSS** on the frontend, **Django REST Framework** on the backend, and **Supabase Postgres** as the database, all wrapped in **Docker** for one-command local development.

> A precision, architectural UI: zero border-radius, sharp surfaces, mature professional palette.
